# Alphabet & Counting Game

An interactive educational game for children to learn the alphabet and practice counting.

## Features

### Alphabet Game
- Interactive alphabet ordering game
- Players click letters in the correct alphabetical sequence (A-Z)
- Visual and audio feedback for correct/incorrect selections
- Fun dinosaur celebration animation with confetti when completed

### Counting Game
- Numbers from 1 to 20
- Audio pronunciation when numbers are clicked
- Visual star display to represent the clicked number
- Confetti celebration when reaching 20

## Technologies Used
- HTML, CSS, JavaScript
- Web Speech API for letter pronunciation
- Canvas Confetti for celebrations
- Audio effects for engagement

## How to Play
1. **Alphabet Game**: Click on letters in alphabetical order (A, B, C, etc.)
2. **Counting Game**: Click on numbers to hear them and see the corresponding number of stars

## Demo
To play the game, simply open the index.html file in a web browser or visit the GitHub Pages site.
